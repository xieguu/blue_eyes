## 功能特性

### 🔆 护眼控制
| 功能 | 说明 |
|------|------|
| **色温调节** | 范围 2000K–8000K，低色温过滤蓝光，适合夜间使用 |
| **亮度调节** | 软件层面调节 Gamma 亮度（30%–100%） |
| **超暗模式** | 在显示器硬件亮度极限之下叠加透明遮罩，深夜更护眼 |
| **昼夜自动模式** | 按 24 小时曲线自动调节色温（白天 6500K → 深夜 2700K） |
| **6 种预设模式** | 常规 / 办公 / 游戏 / 阅读 / 睡眠 / 户外，一键切换并平滑渐变 |

### ⏲ 休息提醒
| 功能 | 说明 |
|------|------|
| **自定义间隔** | 工作时长 5–120 分钟可调 |
| **自定义休息时长** | 10–300 秒可调 |
| **视力训练动画** | 休息时小球沿椭圆轨迹引导眼球转动，放松睫状肌 |
| **强制休息模式** | 前 10 秒锁定"跳过"按钮，确保眼睛真正离开屏幕 |
| **托盘气泡预警** | 休息前 1 分钟弹出气泡提醒，不突然全屏打断 |
| **全屏自动推迟** | 检测到游戏/视频全屏时，自动推迟 5 分钟，附进程白名单过滤 |

### 📊 用眼统计
- 今日用眼总分钟数
- 本次连续用眼时长
- 已休息次数
- 今日用眼目标进度（圆环，480 分钟为上限）
- 近 7 天用眼时长柱状图

### ⚙ 其他
- **系统托盘** — 关闭窗口后后台运行，双击恢复界面
- **开机自启动** — 真实写入注册表 `HKCU\...\Run`，UI 开关联动
- **全局快捷键** — 无需打开界面即可调节（见下表）
- **多显示器支持** — 所有屏幕同步应用 Gamma，热插拔自动补全
- **系统主题色跟随** — 读取 Windows 强调色，UI 配色自动适配
- **HiDPI 自适应** — 4K 屏下界面不模糊
- **配置容错** — 断电损坏自动恢复默认，原子写入防数据丢失

## 快速开始

### 环境要求

- Windows 10 / 11（需要 GDI `SetDeviceGammaRamp` 支持）
- Python 3.10+

### 安装依赖

```bash
pip install PyQt5 pywin32 pynput
```

### 运行

```bash
python care_eyes_pro.py
```

> **提示**：若护眼效果对某些以管理员权限运行的程序（如游戏）无效，请右键 → 以管理员身份运行。

### 打包为独立 .exe（可选）

```bash
pip install pyinstaller
pyinstaller --noconsole --onefile care_eyes_pro.py
```

打包后的 `dist/care_eyes_pro.exe` 可直接运行，无需 Python 环境。

### 护眼控制页

- 右上角**主开关**：关闭后立即还原屏幕，守护定时器暂停（节省 CPU）
- **昼夜自动模式**：开启后色温随时间自动变化，手动滑条仍可临时覆盖
- **超暗模式**：开关 + 强度滑条，多屏幕同步覆盖，鼠标可穿透遮罩正常操作

### 休息提醒页

- 大号倒计时显示距下次休息的剩余时间
- 修改工作/休息时长后点击**应用**立即生效
- **立即测试**：不等计时器触发，直接预览休息效果

### 用眼统计页

- 数据按自然日归零，每天独立记录
- 近 7 天柱状图，今日数据高亮显示

### 设置页

| 选项 | 说明 |
|------|------|
| 开机自动启动 | 勾选后写入注册表，取消勾选后自动删除 |
| 声音提示 | 控制休息提醒时的系统提示音（预留开关） |
| 强制休息 | 休息窗口前 10 秒禁止跳过 |

## 全局快捷键

在任意程序中均可使用，无需切换到 CareEyes 窗口。

| 快捷键 | 效果 |
|--------|------|
| `Ctrl + Alt + ↑` | 亮度 +5% |
| `Ctrl + Alt + ↓` | 亮度 −5% |
| `Ctrl + Alt + →` | 色温 +200K（更冷/更亮） |
| `Ctrl + Alt + ←` | 色温 −200K（更暖/更黄） |
| `Ctrl + Alt + End` | 切换护眼开关（紧急还原） |

> 快捷键依赖 `pynput` 库，若未安装则此功能静默禁用，其他功能不受影响。

## 配置文件

配置保存在用户主目录：

```
C:\Users\<你的用户名>\.care_eyes_pro.json
```

采用**原子写入**（先写 `.tmp` 再 `os.replace`），即使断电也不会损坏原文件。

示例内容：

```json
{
  "temp": 4500,
  "bright": 0.85,
  "rest_interval": 45,
  "rest_duration": 20,
  "force_rest": false,
  "auto_mode": false,
  "super_dim": false,
  "super_dim_alpha": 80,
  "sound_enabled": true,
  "stat_date": "2025-08-01",
  "today_minutes": 132,
  "week_data": {
    "2025-07-26": 310,
    "2025-07-27": 420
  }
}
```

> 手动编辑后重启程序生效。若文件格式错误，程序启动时自动恢复默认值。

## 技术架构

```
care_eyes_pro.py
│
├── _read_system_accent()     读取 Windows 强调色（注册表）
├── _is_admin()               检测当前进程权限
│
├── DisplayManager            多显示器 Gamma 控制
│   ├── _kelvin_to_rgb()      色温 → RGB 比例（Tanner Helland 算法）
│   ├── _build_ramp()         构建 256 级 Gamma 查找表
│   └── apply() / reset()     EnumDisplayMonitors 遍历所有屏幕
│
├── SmoothTransition          平滑色温渐变（ease-out cubic，50ms 步进）
│
├── DimManager                多屏超暗遮罩管理
│   └── SuperDimOverlay       单屏透明遮罩（WindowTransparentForInput）
│
├── EyeExerciseOverlay        视力训练休息窗口
│   ├── 小球动画（16ms 定时器，椭圆轨迹）
│   └── 强制模式（前10秒禁用跳过）
│
├── AnimatedToggle            自绘圆润开关控件
├── RingProgress              自绘圆环进度条
├── BarChart                  自绘7天柱状图
│
└── CareEyesApp               主窗口
    ├── init_timers()         守护(800ms) / 统计(1min) / 倒计时(1s) / 自动模式(1min)
    ├── init_hotkeys()        pynput 全局键盘监听（后台线程）
    ├── nativeEvent()         WM_SETTINGCHANGE / WM_DISPLAYCHANGE / WM_POWERBROADCAST
    ├── _is_fullscreen()      进程名黑白名单 + 窗口尺寸双重判断
    └── _save_settings()      原子写入 JSON
```

### 核心定时器说明

| 定时器 | 频率 | 用途 |
|--------|------|------|
| `guard_timer` | 800ms | 强制重应用 Gamma，抵抗 UAC/驱动重置 |
| `rest_timer` | 用户设定 | 触发休息提醒 |
| `countdown_timer` | 1s | 刷新"距下次休息"倒计时 |
| `stat_timer` | 1min | 累加今日用眼分钟数 |
| `auto_timer` | 1min | 计算并应用昼夜色温曲线 |

## 常见问题

**Q: 护眼效果对某些程序没用？**  
A: 以管理员身份运行 CareEyes Pro。部分游戏或以高权限运行的程序会阻止 GDI Gamma 调用。

**Q: 打游戏时弹出了休息提醒？**  
A: 将游戏进程名添加到代码顶部的 `FULLSCREEN_FORCE_DEFER` 集合，或确认进程名不在 `FULLSCREEN_WHITELIST` 中被误过滤。

**Q: 快捷键不生效？**  
A: 确认已安装 `pynput`（`pip install pynput`）。部分安全软件会拦截全局键盘钩子，需要放行。

**Q: 多显示器中有一块屏没有护眼效果？**  
A: 部分独立显卡驱动会忽略 GDI Gamma。可尝试在显卡控制面板（NVIDIA/AMD）中关闭"数字振动"或"色彩增强"等功能。

**Q: 配置文件在哪里，怎么重置？**  
A: 删除 `C:\Users\<用户名>\.care_eyes_pro.json`，重启程序即恢复默认设置。
